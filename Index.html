Animals API

The Open Animals API

Animals API is a REST full web service that provides various information about animals, both text and images that will make it easier for users to use this web service.

Informations And Images Animals API

Currently over 280,000 posters, updated daily with resolutions up to perfect.

Base URL

Copy This URL And Add Query To Use The AnimalsAPI

http://wwww.animalsapi.com/animals?APIkey=

Query-Table
No Query Required Value Response
1 APIkey Yes ?APIkey=...Your APIkey Json
2 ByName Optional ?Name=... Json
3 ByType Optional ?Type=... Json
4 ByCategory Optional ?Category=... Json
5 ByKelas Optional ?Kelas=... Json
6 ByCategoryAndType Optional ?Category=...&Type=... Json
7 ByCategoryAndKelas Optional ?Category=...&Kelas=... Json
Example

You are required to fill in the APIkey first and then add a query to request to our API
